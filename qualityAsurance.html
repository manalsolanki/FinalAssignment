Quality Assurance

What is quality assurance ?

Quality Assurance consists of different procedure to ensure the quality of the product is provided to the customer as per the requirement. It mainly focuses on making the application efficient and effective as per the coding standards.

Measures Involved

  • Version Control

    This website is developed using Github. Intially it was created on github and the repository was clone to the local computer. By doing this we can edit code install packages and check the code once build successfully it is pushed to github again. Easy to share and maintain the track of project..

  • Adhoc Testing

    • HTML Validator : To check that website is valid and no errors in html.
    • CSS Validator : To check that styling file is structured correctly.
    • Linters :To ensure that there are no errors in my Js file. Also used console.log to make sure code is working as intended.
    • Light house Auditing : Generated the report, almost got 100% for the website just for SEo it was about 89%.
    • aXe Auditing : To check that styling file is structured correctly.

  • Unit Testing

    This type of testing is generally done by dividing the application in individual components. The main aim to check that each component work as indented.

    Performed unit testing with the help of Mocha and Chai Library.In the test folder both the test cases are aviable.

  • Website Deployment

    • Build using VITE Tool
    • Editor used Visual Studio Code
    • Build and Preview on local
    • Final Deployment on Netlify
